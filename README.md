Para acceder a más funcionalidades del sistema, te permite ingresar como administrador. Existen categorías las siguientes categorías de productos: obra gruesa, obra fina, herramientas y maquinaria. Dicho ingreso, cuya sesión se guarda en el Local Storage para poder acceder en diferentes pestañas del navegador

El administrador de llantucha puede agregar productos que persisten en la BD(existe la funcionalidad de vista previa antes de ser agregado) y ver stock de un producto especifico solo como una muestra no desde la BD; se puede leer la descripcion de los productos,su precio y uso. 
Se registran los ids de los productos(BD) con los que interactúa,la hora de ingreso y los nombres de estos productos todo esto en el session storage. Ademas la aplicacion es responsiva parte de las funcionalidades de angular.

PD El código de la interacción con la base de datos está en otro repositorio.
